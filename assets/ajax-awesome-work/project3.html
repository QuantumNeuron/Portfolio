Dynamic Grid Creator
by Peter Gajos
This web application was created as an In-Lab assinment for the WEB222 Web Programming Principles class. This application, allowed a user to create a grid in any desired size and add or delete rows and columns. This assignment used HTML and basic JavaScript library functions including element creation/deletion and procedural logic. Although small, its development took over 6 hours without any help from peers or the instructor.
