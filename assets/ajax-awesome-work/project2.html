SIN Validation Tool
by Peter Gajos
This small program was created for the OOP244 C++ Programming Class. It was meant to validate a real social insurance number. The program demonstrated the essential tenents of C++; Abstraction, Encapsulation and Polymorphism. I learned how to use classes, private and public members and standard output formatting. The program was successfull because it ran the first time after coding, and didn't need any debugging -- a rare occurance.
