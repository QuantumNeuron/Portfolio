Portfolio Website
by Peter Gajos
This reactive and mobile-first website was created as the main component of the Portfolio assignment for EAC397 Business Report writing class. It was meant to be a professional platform to demonstrate my industry specific skills. This project used Photoshop, HTML5, CSS3, SCSS, AJAX, JavaScript and most importantly Bootstrap 4 framework. Beforehand, I did not have any familiarity with the Bootstrap framework. During the creation of this website, I taught myself one of the newest and most popular framworks, Bootstrap 4. The final outcome looks quite professional, and demonstrates successful web development.
